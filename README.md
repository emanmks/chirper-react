# Chirper React

A chirper app built with:

- Laravel
- Jetstream
- Inertia with React
- SQLite
- Typescript

## How to start

- Clone this repo, then:

```bash
docker-compose up -d
```

- Run: `./vendor/bin/sail composer install`
- Run: `./vendor/bin/sail npm install`
- Run: `./vendor/bin/sail npm run dev`
